Very fast yaml parser for Lua

This parser uses libYAML by Kirill Siminov with bindings by Andrew Danforth and some modifications by Gaspard Bucher. libYAML version 0.1.3.

This module is part of the lubyk project. MIT license © Gaspard Bucher 2014.

Installation

With luarocks:

$ luarocks install yaml

Usage example

local data = yaml.parse(some_yaml)

local yaml_string = yaml.dump(some_table)

.VERSION = '1.0.0'

.DEPENDS = {

'lua >= 5.1, < 5.3'

'lub >= 1.0.3, < 1.1'

Uses Lubyk base library

}

Lua table

Lua tables works as sets. In Lua, there is no distinction between an array and a dictionary whereas YAML makes this distinction. When dumping a table, the library has to choose between these two types. Here is the selection rule:

Class methods

.parse (string, safe)

Parse a string containing yaml content and return a table. Uses yaml.Parser internally. If the second argument safe is true, the library will not generate aliases in tables.

.load (path)

Parse the YAML content of the file at path and return a lua table. Uses yaml.Parser internally.

.dump (data)

Dump lua content as YAML.
