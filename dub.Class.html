Class

A class/struct definition.

.__call (lib, self)

This lets sub-classes of dub.Class like dub.Namespace or dub.CTemplate use the same call convention to create new objects.

.new (self)

Create a new class definition. Most of the attributes are passed with self. Usage example:

local class = dub.Class {
  -- self can be a class or db (root)
  db      = self.db or self,
  parent  = parent,
  name    = name,
  xml     = elem,
  xml_headers  = {
    {path = header.dir .. lk.Dir.sep .. elem.refid .. '.xml', dir = header.dir}
  },
}
