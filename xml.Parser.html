  • xml.Parser

Very fast xml parser

The parser does not return information on comments, declarations, doctypes and pi nodes. These are validly parsed but not returned.

This module is part of lubyk project. Install with luarocks or luadist.

$ luarocks install xml    or    luadist install xml

Dump a table to xml example:

local xml = require 'xml'

Parser types

The following constants can be used with new when creating a parser. Depending on the type, the parser behavior is different. The default type is Default.

Default

Default parser type

NonDestructive

Type of blah

Fastest

xxx

.new (type)

Create a new parser. Type flag is optional. Usage example:

local xml = require 'xml'
local parser = xml.Parser(xml.Parser.Fastest)

:parse (str)

Parse an xml string and return a Lua table. See lua/xml for the format of the returned table. Usage:

local data = parser:parse(some_xml_string)
--> lua table
